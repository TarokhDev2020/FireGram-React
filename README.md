
# FireGram-React

How to use firebase and firestore in a react application
## Dependencies

- firebase
- firebase-react
- framer-motion
- motion

## Installation

Install my-project with npm


```bash
  git clone https://github.com/TarokhDev2020/FireGram-React.git
  cd my-project
  npm install
```

make sure to create your application in firebase first
